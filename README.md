# 基于Python实现的QQ机器人

#### 介绍
通过 *Nonebot* 框架和 *gocq* 实现的QQ机器人

#### 软件
需要python环境。且需要Python 3.8 以上版本


#### 安装教程

1.  下载依赖
~~~properties
pip install
~~~
2.  修改配置 config.yml 里的qq号和密码
3.  启动bot.py

#### 使用说明

1.  保证端口不被占用
2.  在配置中不填写账号密码则需要扫码
3.  部署在云端建议更换 session.token
